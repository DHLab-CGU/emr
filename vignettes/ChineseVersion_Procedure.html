產生 ICD-9-PCS 的兩種格式的表格: Decimal 和 Short

供後續 ICD 格式轉換的功能使用 (convertIcdPrDecimaltoShort 和 convertIcdPrShortToDecimal)

ICD-9-PCS

icd9 <- ccsPrICD9$ICD
ICD9PrwithTwoFormat <- ICD9PrGenerateDecimalFormat(icd9)

icd9_Short <- ICD9PrwithTwoFormat$Short[1:10]
# c("E9500", "E9501", "E9502", "E9503", "E9504", "E9505", "E9506", "E9507", "E9508", "E9509")
icd9_Decimal <- ICD9PrwithTwoFormat$Decimal[1:10]
# c("E950.0", "E950.1", "E950.2", "E950.3", "E950.4", "E950.5", "E950.6", "E950.7", "E950.8", "E950.9")

ICD-10-PCS只有short的格式

ICD-9-PCS and ICD-10-PCS code 兩種格式的轉換: Short <-> Decimal

進行後續分組功能之前, 根據使用者使用的 ICD 格式進行格式的統一化, 分別兩種功能可以將 ICD 統一轉換為 Short 或 Decimal 的格式

將 ICD-PCS code 的格式統一由 Short 轉成 Decimal

# ICD-9
convertIcdPrShortToDecimal(icd9_Short)$Decimal
# c("E950.0", "E950.1", "E950.2", "E950.3", "E950.4", "E950.5", "E950.6", "E950.7", "E950.8", "E950.9")

# ICD-10
convertIcdPrShortToDecimal(icd10_Short)$Decimal
# c("A15.0", "A15.4", "A15.5", "A15.6", "A15.7", "A15.8", "A15.9", "A17.0", "A17.1", "A17.81")

將 ICD-PCS code 的格式統一由 Decimal 轉成 Short

# ICD-9
convertIcdPrDecimaltoShort(icd9_Decimal)$Short
# c("E9500", "E9501", "E9502", "E9503", "E9504", "E9505", "E9506", "E9507", "E9508", "E9509")

# ICD-10
convertIcdPrDecimaltoShort(icd10_Decimal)$Short
# c("A150", "A154", "A155", "A156", "A157", "A158", "A159", "A170", "A171", "A1781")
